## Команды Django

- Запустить проект:
  ```bash
  ./manage.py runserver
  ```

- Создать суперпользователя:
  ```bash
  ./manage.py createsuperuser
  ```
admin
12345
- Создать миграции:
  ```bash
  ./manage.py makemigrations
  ```

- Применить миграции к базе данных:
  ```bash
  ./manage.py migrate
  ```
